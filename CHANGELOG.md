# 更新日志 / Changelog

本项目遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/) 风格，版本号遵循语义化版本。

## [2.0.0] - 2026-06-10 · 全天候 + 分钟级 + 杠杆 + 跨市场 + 新 UI

### 新增
- **全天候按时段调度**：按美东时区（自动处理夏/冬令时与节假日）切换盘前 / 盘中 / 盘后 / 隔夜 / 休市，各时段独立扫描节奏。
- **分钟级日内信号**：VWAP、日内 RSI、近 5/15 分钟涨速、量能脉冲、日内位置；事件检测新增急跌急涨、跌破/收复 VWAP、跳空、日内新高/新低、放量脉冲。
- **隔夜代理 + 开盘前预警**：个股/ETF 隔夜不交易时，监控指数期货（ES=F/NQ=F）与亚洲股市（KOSPI/日经/恒指/上证）；持有的杠杆 ETF 挂钩市场大动时提前发「开盘前预警」（如 KOSPI 暴跌 → KORU）。
- **杠杆 ETF 支持**：内置常见杠杆倍数，按倍数放大日内阈值；AI 强调杠杆放大与每日复利损耗、不宜长持。
- **跨市场宏观联动**（`macro.py`）：期货 + 亚洲盘喂入 AI；杠杆 ETF 自动挂钩标的市场（KORU→韩国、SOXL→费城半导体…），`UNDERLYING_MAP` 可自定义。
- **AI 多空辩论升级**：技术面 + 日内 + 宏观 + 新闻一并喂入，输出多头/空头论据 + 倾向 + 关键位 + 风险 + 新手提醒。
- **全新 Web 盯盘台**：采用 huashu-design「FT × 彭博终端」编辑风（浅底报纸感 + 1px 规则线 + 等宽字 + 涨红跌绿，零 emoji/紫渐变）；分组看板（大盘/期货/亚洲/个股/杠杆ETF）+ 宏观条 + AI 信号流 + 个股**分钟 K 线图**（lightweight-charts）+ mini 走势线。
- **新 API**：`/api/ohlc`（K线）、`/api/macro`（跨市场）、`/api/spark`（走势线）；`/api/quotes` 增加时段信息。
- **更多可配置项**：杠杆清单、期货/亚洲清单、挂钩映射、日内/隔夜扫描间隔、日内急跌阈值、隔夜异动阈值、盘前盘后开关等，全部在 `/settings` 可改。

### 改进 / 修复（经多智能体对抗式代码评审，确认并修复 15 项）
- 推送失败不再占用冷却（避免最该提醒时被静默吞掉）。
- 修复启动竞态（双击启动可能起两个线程）。
- 行情空结果加入负缓存，降低 yfinance 限流风险。
- 识别美股节假日（节假日不再误判为交易时段）。
- 保存设置后运行中的盯盘线程热重载配置，关注列表/阈值即时生效。
- 公开路由对标的代码做白名单校验；会话 Cookie 加固（HttpOnly + SameSite=Lax）。
- 飞书卡片不再出现字面 "None"；新闻链接限制 http(s) scheme。
- 复盘触发改为跨越式判定，避免窗口过窄漏触发。
