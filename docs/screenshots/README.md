# 截图放这里

主 README 与 `docs/USAGE.md` 已预先引用以下文件名，**把对应 PNG 放进本目录并提交即可自动显示**：

| 文件名 | 内容 | 怎么拍 |
|---|---|---|
| `board.png` | 盯盘台主页（分组看板 + 宏观条 + 信号流）| 浏览器开 `http://localhost:5005`，整页截图 |
| `detail.png` | 个股详情页（分钟 K 线图 + 信号史 + 新闻）| 点任一标的卡片进详情页截图 |
| `settings.png` | 设置页（关注列表 / 持仓 / AI / 飞书）| 登录后进「设置」截图 |
| `feishu.png` | 飞书收到的盯盘卡片 | 手机/飞书桌面端截图一张信号卡 |

> 建议宽度 ≥ 1200px。截好后：`git add docs/screenshots/*.png && git commit -m "docs: 截图" && git push`。
> 在此之前，README 顶部的 `docs/preview.svg` 是手绘示意预览，能先让访客看到界面风格。
